01
🖥️
Pinning
Az elem position: sticky-vel vagy GSAP pin:true-val rögzül. A ScrollTrigger figyeli a görgetési pozíciót és automatikusan kezeli a helyettesítő elemet (spacer).
pin: true
02
📍
Trigger pontok
A start: "top top" határozza meg mikor kezd rögzülni. Az end: azt mondja meg, mikor engedik el. Közöttük teljesen a görgetési pozíció vezérli.
start: "top top"
03
🎬
Scrub
scrub: true esetén az animáció közvetlenül a scrollhoz van kötve – nincs "időzítés", az egér pozíciója vezérel mindent. scrub: 1 simítást ad hozzá.
scrub: 1
04
⚡
Markers debug
Fejlesztés közben markers: true mutatja a trigger és end pontokat a viewport-on – így pontosan látod mikor aktivál a ScrollTrigger.
markers: true

Kód vázlat

CSS sticky-vel implementálva, GSAP-nélkül is működő verzió.

// GSAP verzió:
gsap.registerPlugin(ScrollTrigger);

ScrollTrigger.create({
  trigger: ".pin-container",
  start: "top top",
  end: "bottom bottom",
  pin: ".pin-left",
  pinSpacing: false,
});

// CSS verzió (egyszerűbb):
.pin-left {
  position: sticky;
  top: 0;
  height: 100vh;
}
