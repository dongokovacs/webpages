02 – Kártya hover effektek

Mouse-tracking effektek

Spotlight, border glow conic-gradient és 3D tilt – vidd az egeret a kártyák fölé.

✦
Spotlight Card
A radial gradient követi az egér pozícióját a kártyán belül – puha belső fényt kelt.
🌀
Border Glow
Conic-gradient rotál a kártya körül hover esetén – iridescent szegély effekt.
🎯
3D Tilt
Perspektíva-alapú 3D billentés – az egér pozíciója dönti el a X/Y rotációt.
// Magnetic button
btn.addEventListener('mousemove', e => {
  const r = btn.getBoundingClientRect();
  const x = e.clientX - r.left - r.width / 2;
  const y = e.clientY - r.top - r.height / 2;
  const strength = 0.35;
  btn.style.transform =
    `translate(${x*strength}px,${y*strength}px)`;
});
btn.addEventListener('mouseleave', () => {
  btn.style.transform = 'translate(0,0)';
  btn.style.transition = 'transform .6s cubic-bezier(.16,1,.3,1)';
});

// 3D Tilt
card.addEventListener('mousemove', e => {
  const r = card.getBoundingClientRect();
  const x = (e.clientX - r.left) / r.width - 0.5;
  const y = (e.clientY - r.top) / r.height - 0.5;
  card.style.transform =
    `perspective(800px) rotateX(${y*-15}deg) rotateY(${x*15}deg)`;
});
