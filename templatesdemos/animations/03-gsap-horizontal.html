Hogyan működik?

A wrapper rögzítve van (pin: true). A track vízszintesen tolódik el – a görgetési távolság egyenlő a track szélességével mínusz a viewport szélességével.

gsap.registerPlugin(ScrollTrigger);

const track = document.getElementById("h-track");
const distance = track.scrollWidth - window.innerWidth;

gsap.to(track, {
  x: -distance,
  ease: "none",
  scrollTrigger: {
    trigger: "#h-section",
    start: "top top",
    end: () => `+=${distance}`,
    pin: true,
    scrub: 1,
    invalidateOnRefresh: true,
  },
});
